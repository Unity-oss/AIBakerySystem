# 🍰 Tastyz Bakery AI System

## Project Overview

Tastyz Bakery AI System is an intelligent, full-stack web application built for a real bakery business. The system solves the operational challenge of managing customer inquiries, product discovery, order processing, and backend business analytics — all powered by Large Language Models (LLMs). It combines a **RAG-powered customer support chatbot** (backed by the bakery's own FAQ and price list), a **product recommendation engine**, an **AI order-processing agent**, and **backend automation agents** for sales analysis and email summarization. The application is orchestrated with **LangGraph**, sand is served through a **Django** web framework.

## Architecture

```
Customer Browser
      ↓
Django Web App (4 pages: Home, Products, Order, AI Chat, Recommendations)
      ↓
LangGraph Agents
  ├── RAG Support Agent  → ChromaDB (FAQ + Price List embeddings)
  ├── Order Agent        → Django DB + Excel export
  ├── Recommendation AI  → OpenAI + Product catalog
  └── Backend Agents (Celery)
        ├── Sales Analysis Agent
        └── Email Summarization Agent
```

## Tech Stack

| Layer | Technology |
|-------|------------|
| Web Framework | Django 4.2 |
| LLM Orchestration | LangGraph + LangChain |
| LLM Provider | OpenAI (GPT-4o-mini) |
| Vector Store | ChromaDB |
| Embeddings | OpenAI text-embedding-3-small |
| Prompt Templates | Jinja2 |
| Background Tasks | Celery + Redis |
| Data Export | pandas + openpyxl |
| Linting | Ruff |
| Logging | Python JSON Logger |

## Features

- 🤖 **AI Chatbot** — RAG-powered assistant answers questions about products, delivery, pricing, and policies
- 🎂 **Product Recommendations** — AI suggests cakes and pastries based on customer needs
- 📋 **Order Processing** — Form-based ordering with automatic Excel export and calendar scheduling
- 📊 **Sales Analysis** — Daily automated sales report generated by an AI agent
- 📧 **Email Summarization** — AI summarizes customer emails for the baker
- 🔐 **Secure** — All API keys managed via `.env`, input validation on all forms

## Setup

```bash
# 1. Clone and install
pip install -r requirements.txt

# 2. Configure environment
cp .env.example .env
# Fill in your API keys in .env

# 3. Build knowledge base & run migrations
python manage.py migrate
python manage.py build_knowledge_base
python manage.py seed_products

# 4. Run development server
python manage.py runserver

# 5. (Optional) Start Celery for background agents
celery -A tastyz_project worker -l info
celery -A tastyz_project beat -l info
```

## Ethical Considerations

- **Data Privacy**: Customer orders stored locally; no PII sent to external APIs beyond necessary context
- **Bias Mitigation**: Product recommendations draw from the full catalog without preference
- **Transparency**: Users are informed they are talking to an AI assistant
- **Rate Limiting**: API calls are managed to prevent abuse and runaway costs
